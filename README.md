# 🧩 Wordle Game

👩‍🔧 Building in progress...


## 🚀 How to play
Git clone this repo, `cd` into its folder and follow those steps.

1. Install dependencies
```shell
bun install
```

2. Run app locally
```shell
bun run dev
```


## 🍱 Stack
This app is built with Vite using TypeScript, React and Tailwind CSS and runs on Bun.
